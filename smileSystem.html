Smile System

O Smile System é um sistema projetado para atender as necessidades de clinicas odontológicas. Foi desenhado para separar as funcionalidades em módulos, facilitando a expansão de novos requisitos e atualizações. Atualmente o sistema conta com o módulo de agendamento de consulta, controle de profissionais e pacientes. Futuras versões do software serão desenvolvidas em parceria com os clientes, adaptando-se as necessidades individuais de cada um.
